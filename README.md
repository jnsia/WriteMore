# 글로모여

## 프로젝트 URL

**https://test-writemore.web.app/**

Firebase Hosting을 통해 배포하였습니다.

## 기획 의도

대구광역시에서 진행되던 글 쓰기 및 독서 모임에 대한 정보를 통합하여 유저들에게 제공하며, 이를 통해 "글로모여"라는 키워드를 브랜딩하고자 했습니다.

## 프로젝트 기간

2023.03 ~ 2023.04 (6주)

## 개발 환경

- HTML5
- CSS3
- NodeJS 18.12.1
- Firebase v9
    - Firebase Hosting
    - Firebase Realtime Database
    - Firebase FireStore
